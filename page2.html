Snowy Train 3/2/15

Written in Java

One of my last class-oriented java projects (completed as a no-specs extra credit assignment), SnowyTrain created many Snowflake objects, all of which inherited gravity and windspeed from the environment, which varied over time. A random modifier allowed each flake to have slightly different speeds in relation to the background wind, giving a more realistic feel. A "train" could move in the background, going offscreen for a while before returning to the left. All in all, this project was the height of any graphics project I made.

Map Generator 12/10/13

Written in C++

This is my first real code project. It runs on Unix and produces maps. The tiles are 100% customizable (color, char, chance attributes, etc.), and can be edited within the program before producing another map of custom size. This is one of the more simple presets, but there were cases where I had mountains, swamps, cities, and other landmarks, creating a pretty cool Dwarf-Fortress look. The generation is simple but effective. All in all, there were ~500 lines of code. I made a Windows equivalent as well, but can't seem to find it.
