Hotel Reservation Maker 2/12/16 - 2/19/16 (+ minor updates)

Written in Java

Created for a potential employer, this project was both a learning experience and a showcase of what I can do in a week's time. I plan to make some final polishing changes, but it is largely complete. It uses a local Access database to display available hotel rooms, taking into account existing reservations. It also stores user credentials for some basic login capability. Midway through the week-long trial, I took a big gamble overhauling the entire program to switch to an actual DB (okay, Access isn't a real professional database). It paid off, and I continued to try out a lot more new stuff (more sophisticated GUI layouts, prepared SQL statements, various Java 8 objects, dynamically changing listeners, etc.) for the first time. Probably the most progression I've had over the course of a single project. There is a lot that could be improved internally, but I'd rather spend time on applying the improvements on future code. Externally, the program is hard to break and performs as expected. It is officially closed. New users can be created, but the default user ajsnyde has the password "1234".

Snowy Train 3/2/15

Written in Java

One of my last class-oriented java projects (completed as a no-specs extra credit assignment), SnowyTrain created many Snowflake objects, all of which inherited gravity and windspeed from the environment, which varied over time. A random modifier allowed each flake to have slightly different speeds in relation to the background wind, giving a more realistic feel. A "train" could move in the background, going offscreen for a while before returning to the left. All in all, this project was the height of any graphics project I made.
