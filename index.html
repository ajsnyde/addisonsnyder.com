Snowy Train 3/2/15

Written in Java

One of my last class-oriented java projects (completed as a no-specs extra credit assignment), SnowyTrain created many Snowflake objects, all of which inherited gravity and windspeed from the environment, which varied over time. A random modifier allowed each flake to have slightly different speeds in relation to the background wind, giving a more realistic feel. A "train" could move in the background, going offscreen for a while before returning to the left. All in all, this project was the height of any graphics project I made.
